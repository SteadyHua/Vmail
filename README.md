# 关于项目

这是我的毕业设计(2018)，邮件客户端

目前尚未完成，会继续修改

## 使用的相关模块

* 用electron-vue构建项目
* 用node-imap模块接收邮件
* 用nodemailer发送邮件
* 用element-ui组件库
* 用lowdb做本地数据存储
* 用iconv-lite、quoted-printable、utf8等处理编码
* 用vue-quill-editor做富文本编辑器

``` bash
npm run dev    # 调试运行，localhost:9080

npm run build  # 打包
```
